  • 总结

    Markdown是一门比HTML更简单的标记语言，其主要用于日常写作，博客文章等。最终通过相应的编辑器换HTML用于页面渲染，会极大提高自己文章的整体性。

    优点：
    1.使用简单，书写流畅，作者无须关注排版格式
    2.Markdown一般都具有双栏模式，可以直观的预览编写结果，书写错误容易发现
    3.导出简单，只要是markdown编写的源文本，拷贝到任何一个其他markdown平台基本都是有效 的，而拷贝富文本的编辑内容，通常都需要进行一些额外的处理
